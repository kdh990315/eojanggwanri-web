# Product Brief

## Purpose

`어장관리` 웹사이트는 초보 낚시인이 검색으로 들어와 "사진으로 물고기 이름을 찾는" 문제를 바로 해결하고, 자연스럽게 어장관리 앱의 조과 기록과 낚시 노트 가치를 이해하게 만드는 소개 사이트다.

첫 화면의 중심은 일반 앱 홍보 문구가 아니라 AI 어종 판별 체험 진입이다. 사용자는 메인 페이지에서 핵심 가치를 이해하고, 상단 메뉴와 메인 CTA를 통해 `/ai-species` 페이지로 이동할 수 있어야 한다.

## Target User

- 검색으로 유입된 초보 낚시인
- 잡은 물고기 이름을 사진으로 확인하고 싶은 사용자
- 낚시 사진, 위치, 날짜, 물때, 날씨를 조과 기록으로 남기고 싶은 사용자

## Core Flow

```text
검색 유입
  -> 어장관리 소개 페이지
  -> AI 어종 판별 CTA
  -> 사진 업로드
  -> 영상 광고 노출
  -> 광고 skip 또는 시간 종료
  -> AI 어종 후보 확인
  -> 조과 기록 앱 가치 이해
```

## Confirmed Decisions

- 웹 체험은 로그인 없이 AI 어종 판별을 사용할 수 있다.
- 비로그인 판별 사진은 저장하거나 보관하지 않는다.
- 광고는 보상형 광고가 아니다.
- 이미지 업로드 후 영상 광고를 노출하고, 사용자가 skip하거나 광고 시간이 끝나면 AI 어종 판별을 시작한다.
- AI 어종 판별은 기존 앱 Supabase Edge Function을 그대로 재사용한다.
- 앱 다운로드 링크는 소개페이지에서만 노출한다.
- 소개페이지에는 GitHub 링크를 노출하지 않는다.
- 문의, 오류 제보, 아이디어 제안은 Google Form으로 받는다: `https://docs.google.com/forms/d/e/1FAIpQLSexQf4Mtne8w5iEjDjW-06VGYFV1cdGLUDX8Q_8gT6kLvRCfQ/viewform?usp=dialog`
- 어종 생태 정보와 금어기/금지체장 정보의 공식 출처는 Supabase DB에 저장된 데이터를 사용한다.

## Product Boundaries

- 메인 페이지는 소개페이지다.
- AI 어종 판별 기능은 핵심 진입점이지만 메인 페이지에서 판별 UI 전체를 구현하지 않는다.
- 고급 통계는 MVP 이후 확장 가치로 표현한다.
- 커뮤니티, 랭킹, 결제, 고급 통계 화면을 주요 기능처럼 노출하지 않는다.
- AI가 항상 정확하다는 인상을 주지 않는다.
